# 🍕 Pizza Sales Dashboard | Power BI

## 📊 Project Overview

The **Pizza Sales Dashboard** is an interactive Business Intelligence project built using **Power BI**. It provides comprehensive insights into pizza sales performance by analyzing revenue, orders, customer purchasing patterns, and product performance. The dashboard enables stakeholders to monitor key business metrics and make data-driven decisions through dynamic visualizations.

---

## 🎯 Project Objectives

* Analyze overall sales performance.
* Monitor revenue and order trends over time.
* Identify best-selling and least-selling pizzas.
* Compare sales across pizza categories and sizes.
* Track key business KPIs to support decision-making.

---

## 📈 Key Performance Indicators (KPIs)

* 💰 Total Revenue
* 🛒 Total Orders
* 🍕 Total Pizzas Sold
* 📊 Average Order Value
* 📦 Average Pizzas per Order

---

## 📌 Dashboard Features

* Interactive and dynamic dashboard
* Daily and monthly sales trend analysis
* Category-wise sales analysis
* Size-wise sales analysis
* Best-selling and worst-selling pizzas
* Revenue distribution by pizza category
* Filters and slicers for easy data exploration
